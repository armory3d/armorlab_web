AI-Powered Texture Authoring

ArmorLab is a stand-alone software designed for AI-powered texture authoring. Generate PBR materials by drag & dropping your photos.

PBR Textures

Extract base color, height, normal map, occlusion and roughness textures from a photo using a pre-trained deep neural network.

    Node Based

    Work fast with the convenience of nodes in a non-destructive environment. Build node graphs to automatically process photo batches.

    Tiling

    Use the tiling node to turn your photos into seamless textures.

      Ray Tracing

      In development - author materials in a viewport with true light simulation. Experimental builds featuring path-traced viewport rendering with Direct3D12 and Vulkan are now available. ArmorLab is powered by an in-house 3D rendering engine to achieve the best experience.

      Inpaint

      Use the inpaint node to fill in texture details or remove unwanted parts from a photo by painting a mask.

        Runs Everywhere

        ArmorLab runs on Windows, Linux, macOS and Android. iPadOS support is in development. On desktop, ArmorLab is a portable application with no installation and near-instant start up. Enhance any part of the software through plugins with god-mode powers.
